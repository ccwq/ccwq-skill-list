# Nano Prompt - AI 图像提示词生成

## 核心原则

1. **分层结构**：提示词按逻辑层级组织，而非堆砌描述
2. **细节桶分类**：光线、氛围、环境、相机、颜色、纹理、构图分开描述
3. **少即是多**：风格锚点选 1-2 个，避免贪多
4. **相机专业术语**：使用电影级相机参数而非模糊描述
5. **微细节触发**：1-3 个词的微细节关键词可触发超写实模式

## 提示词结构模板

输出完整的 YAML 格式提示词，结构如下：

`yaml
创意: 核心主题/主体描述
光线: 光线类型、色温、方向
氛围: 情感氛围关键词
环境: 背景、场景设置
相机: 焦距、光圈、拍摄角度
颜色: 主色调及情感关联
纹理: 材质描述
风格: 风格锚点（1-2个）
细节: 微细节关键词（1-3词触发）
负面: 需要避免的元素
`

## 使用流程

### 1. 解析用户输入
- 提取核心创意/主体
- 识别想要的光线、氛围、风格
- 确定是否需要特定相机参数

### 2. 应用技巧填充

**光线选项**：
- 柔和霓虹、硬光、体积光、背光、边缘光

**相机参数**：
- 35mm、50mm、微距、f1.4-f16 光圈
- 低角度、高角度、俯拍、仰拍
- 变形镜头、运动模糊

**微细节**（超写实触发词）：
- 尘埃颗粒、胶片颗粒、指纹痕迹、环境雾气、冰冷反射

**风格锚点**：
- 吉卜力工作室、写实摄影、赛博朋克、皮克斯着色
- 韦斯·安德森对称、文艺复兴油画、超写实

**情感线索**：
- 孤独、温馨、紧张、怀旧、平静

**负面提示**：
- 无变形、无多余肢体、无过饱和、无卡通感

### 3. 组合输出

按分层模板组合所有元素，生成 YAML 格式输出。

## 输出格式

- 语言：中文
- 格式：YAML
- 嵌套层级：最多 3 层
- Key：中文
- Value：中文描述，关键词可保留英文专业术语
